
# Week 6 Labs - IT5016 Assessment 3

This folder contains Python lab exercises for Week 6 of IT5016, focusing on object-oriented programming (OOP) concepts, inventory management, and library system simulation. These exercises help students understand classes, objects, methods, and program logic while practicing Python programming.



## Folder Contents

InventoryManagement.py  
A simple inventory program that manages items with a unique Item ID, quantity, and price. It calculates the total value of each item. Demonstrates use of functions, global variables, and user input.

LibrarySystem.py  
Implements a basic library system using classes: Book, Member, and Library. Users can add books, register members, and borrow or return books. Demonstrates object-oriented relationships and how object states are managed.

oop.py  
Demonstrates object-oriented programming with Account, Customer, and Transaction classes. Supports deposit and withdrawal operations while keeping track of account balances. Useful for understanding class interaction and method calls.

README.md  
This file, providing an overview of the folder, its contents, and instructions for running the programs.


## Detailed Description of Each Program

InventoryManagement.py  
- Uses a global itemID to assign unique IDs to items.  
- Functions include adding new items with user input and calculating the total price.  
- Demonstrates arithmetic operations, input handling, and function usage.  
- Example: adding an item with quantity 10 and price 5 calculates total price 50.

LibrarySystem.py  
- Contains three classes: Book (title, author, availability), Member (name and borrowed books), and Library (manages books and members).  
- Demonstrates OOP concepts like encapsulation, class interaction, and method usage.  
- Example flow: add books, register members, borrow a book, and display current library status.

oop.py  
- Contains classes representing a banking scenario: Account (account number and balance), Customer (associated with an account), and Transaction (handles deposits and withdrawals).  
- Demonstrates class interaction, modifying object states, and transaction logic.  
- Example flow: create account and customer, perform deposit and withdrawal, display updated account information.

## How to Run the Programs

1. Make sure Python 3.x is installed.  
2. Navigate to this folder in your file explorer or terminal.  
3. Run each program by opening it in an IDE or running it with Python.  
4. Follow the prompts in the program to enter data as requested.

## Learning Outcomes

By completing these labs, students should be able to:  
- Understand the basics of object-oriented programming in Python.  
- Create and use classes and objects effectively.  
- Use functions and methods to organize code logically.  
- Handle user input and manage simple data.  
- Understand program flow in applications like inventory management, library systems, and basic banking transactions.
